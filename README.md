# Taiwan ubike map - 台灣ubike地圖

## About
F2E: Vue.js 3.0  
CSS: BootStrap, SASS  
Map API: leaflet, an open street map  
Open source: Taipei ubike open data  
Ajax: axios, vue-axios

## Features
* Babel  
* CSS Pre-processors  
* Linter  
* Sass/SCSS (with node-sass)  
* Airbnb Lint  

## How to run
1. run on local  
```
npm run serve
```
2. Go to http://localhost:8080
